## What you can build

Loomly is API-first. The dashboard, CLI, MCP server and third-party
integrations (Zapier, Make, n8n) all talk to the same REST endpoints and
operate on the same resources.

## Architecture

Everything routes through a single core:

```text
        Website / Calendar UI ─┐
                CLI ───────────┼──▶  Public REST API  ──▶  Core logic  ──▶  Database
                MCP ───────────┤        (/api/public/v1)
        Zapier / Make / n8n ───┘
```

There is no separate "agent API": humans and agents call the same endpoints
with the same auth.
